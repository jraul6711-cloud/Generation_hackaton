# Generation_hackaton

Este repositorio contiene el desarrollo colaborativo de una landing page realizada como parte del Hackathon número 1 del programa Full Stack Java Developer – Generation CH61.

El proyecto fue ejecutado por un equipo de cinco integrantes, quienes trabajaron de manera estructurada utilizando ramas individuales y procesos de integración mediante Pull Requests. La landing page fue desarrollada con HTML, CSS, Bootstrap y JavaScript, asegurando un diseño responsivo, funcional y alineado a las buenas prácticas de desarrollo web.

## Contribuciones por integrante

- **Emmanuel – Encabezado (Header)**  
  Responsable de la estructura y diseño del encabezado principal, incluyendo la barra de navegación, enlaces y comportamiento responsivo.

- **Adrián – Sección de Beneficios**  
  Encargado de la sección informativa que comunica los principales beneficios y valores agregados del servicio, con enfoque en diseño visual y experiencia del usuario.

- **Maleny – Sección de Pruebas / Evidencias**  
  Desarrolló la sección destinada a mostrar ejemplos, productos, testimonios o elementos visuales que refuerzan la confiabilidad del servicio.

- **Rafael – Formulario de Cotización**  
  Implementó el formulario principal del sitio, incluyendo estructura, estilos personalizados, validación con JavaScript y retroalimentación visual al usuario.

- **Erick – Footer**  
  Responsable del diseño y la implementación del pie de página, incluyendo información complementaria, enlaces relevantes y coherencia visual con el resto del sitio.

Este proyecto demuestra la capacidad del equipo para coordinar esfuerzos, distribuir responsabilidades y entregar una solución web integral dentro de un entorno colaborativo.
